# Novti Landing Page - Build Instructions

## Project Overview
Build a modern, animated single-page website for Novti - a nonprofit fundraising platform. The design should mirror the StoryDoc reference style with smooth scroll animations, interactive elements, and a professional aesthetic.

---

## Design Style & Visual Guidelines

### Color Palette
- **Primary**: `#F23861` (Pink/Magenta)
- **Dark**: `#121212` (Charcoal)
- **Accent**: `#F9DDC6` (Soft Beige)
- **Burgundy**: `#45171A` (Deep Burgundy for contrast sections)
- **Backgrounds**: White, `#F8F8F8` (Light Gray)

### Typography
- **Font Family**: Noto Sans (Google Fonts)
- **Headings**: Bold, 48-72px
- **Subheadings**: Semi-bold, 24-36px
- **Body Text**: Regular, 16-18px
- **Small Text**: 14px

### Layout Principles
- Full-width sections with contained content (max-width: 1200px)
- 50/50 split layouts for hero sections
- Responsive grid system (2-4 columns for cards)
- 8-40px spacing between elements
- Mobile-first responsive design

## JavaScript Components & Libraries

### Essential JavaScript Functionality

#### 1. Scroll Animation Library
**Recommended**: AOS (Animate On Scroll) or ScrollReveal.js
```javascript
// Initialize AOS
AOS.init({
  duration: 1000,
  once: true,
  offset: 100
});
```

**Usage**:
- Fade-in effects for sections
- Slide-in animations for cards
- Stagger animations for lists

#### 2. Animated Counters
**Library**: CountUp.js
```javascript
// Example for conversion rate counter
const counter = new CountUp('conversion-stat', 35, {
  prefix: '+',
  suffix: '%',
  duration: 2.5
});
```

**Apply to**:
- "+18-35%" conversion stat
- "25+" organizations
- Financial projection numbers

#### 3. Chart Visualizations
**Library**: Chart.js
```javascript
// Donut Chart for Market Opportunity
const marketChart = new Chart(ctx, {
  type: 'doughnut',
  data: {
    labels: ['Fundraising', 'CRM', 'CMS'],
    datasets: [{
      data: [400, 137, 41],
      backgroundColor: ['#F23861', '#45171A', '#F9DDC6']
    }]
  },
  options: {
    animation: {
      animateRotate: true,
      animateScale: true
    }
  }
});
```

**Charts needed**:
- Market Opportunity donut chart
- Number of Clients bar chart
- ARR bar chart

#### 4. Smooth Scrolling
```javascript
// Smooth scroll for anchor links
document.querySelectorAll('a[href^="#"]').forEach(anchor => {
  anchor.addEventListener('click', function (e) {
    e.preventDefault();
    document.querySelector(this.getAttribute('href')).scrollIntoView({
      behavior: 'smooth'
    });
  });
});
```

#### 5. Intersection Observer (for viewport detection)
```javascript
// Trigger animations when elements enter viewport
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('animate');
    }
  });
});

document.querySelectorAll('.animate-on-scroll').forEach(el => {
  observer.observe(el);
});
```

#### 6. Parallax Effect (optional enhancement)
**Library**: Rellax.js or custom implementation
```javascript
// Simple parallax for hero background
window.addEventListener('scroll', () => {
  const scrolled = window.pageYOffset;
  const parallax = document.querySelector('.parallax-bg');
  parallax.style.transform = `translateY(${scrolled * 0.5}px)`;
});
```

#### 7. Modal/Popup for CTA
```javascript
// Simple modal for "Book a Demo"
const modal = document.querySelector('.modal');
const ctaButton = document.querySelector('.cta-button');

ctaButton.addEventListener('click', () => {
  modal.classList.add('show');
});
```

#### 8. Timeline Animation
```javascript
// Draw connecting line progressively
const timeline = document.querySelector('.timeline-line');
window.addEventListener('scroll', () => {
  const timelineProgress = calculateProgress();
  timeline.style.height = `${timelineProgress}%`;
});
```

#### 9. Logo Hover Effects
```javascript
// Grayscale to color on hover
const logos = document.querySelectorAll('.client-logo');
logos.forEach(logo => {
  logo.style.filter = 'grayscale(100%)';
  logo.addEventListener('mouseenter', () => {
    logo.style.filter = 'grayscale(0%)';
  });
  logo.addEventListener('mouseleave', () => {
    logo.style.filter = 'grayscale(100%)';
  });
});
```
